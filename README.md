# レビューアプリ
 
投稿者がオススメしたものに対して評価するアプリ
 
## 簡単な説明
 
このアプリは漫画の評価をするアプリです。  
投稿者は自分がオススメしたい漫画を評価付きで投稿します。  
投稿者以外はそれに対して口コミを書くことができます。
